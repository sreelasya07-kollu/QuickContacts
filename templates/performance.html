Data Structure Explanation

📋 Python List

Stores contacts sequentially. Search requires iterating through every contact until a match is found.

Time Complexity: O(n)

🗂️ Python Dictionary

Stores contacts with hash keys for instant access. Lookup by key is nearly constant time.

Time Complexity: O(1)

🔄 Python Queue (deque)

Stores the last 5 searched contacts in FIFO order. Used for recent search history.

Max Size: 5 items
