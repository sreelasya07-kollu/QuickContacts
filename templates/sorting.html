How Sorting Works

🫧 Bubble Sort

Repeatedly compares adjacent elements.

Easy to understand and implement.

Best for: Learning algorithm basics.

🔀 Merge Sort

Uses a divide-and-conquer approach.

Efficient for large datasets.

Best for: Large arrays — performs significantly better than Bubble Sort as input size increases.

💡 Why Compare Them?

Bubble Sort has O(n²) time complexity while Merge Sort has O(n log n).

As the array grows, Merge Sort stays much faster — you can see the difference in the execution times above.
